# 뽀숑이 관리 프로젝트 🐶

2026년 2월 28일생 비숑 프리제 **뽀숑**이의 건강 상태와 교육·훈련을 기록하고,
LLM의 도움을 받아 관리하기 위한 프로젝트입니다.

## 폴더 구조
```
for_bbosyong/
├─ CLAUDE.md            # 뽀숑이 핵심 요약 카드 (항상 최신 유지)
├─ journal/             # 일상 기록 (월별 파일, 날짜별 메모)
├─ health/              # 몸무게 추이 등 건강 데이터
├─ medical/             # 접종·구충·병원 일정
└─ training/            # 개월별 훈련 로드맵 + 수행/회고 로그
```

## 사용 방법
- **기록하기**: "오늘 뽀숑이 ○○했어" 라고 말하면 그 달 `journal` 파일에 추가됩니다.
- **문의하기**: 건강·사료·훈련 등 궁금한 점을 물어보면 현재 상태(CLAUDE.md)를
  바탕으로 답해드립니다.
- **훈련받기**: `training/roadmap.md`의 월령별 추천을 수행하고 `training/log.md`에
  회고를 남기면, 다음 추천에 반영됩니다.

## 모바일에서 쓰기 (목표)
모든 기록을 마크다운 텍스트로 유지하므로, 추후 이 폴더를 git 저장소로 만들어
원격(GitHub 등)에 연결하면 밖에서도 Claude 앱/claude.ai로 접속해
언제든 기록·문의할 수 있습니다.
