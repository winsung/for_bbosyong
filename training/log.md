# 훈련 수행 · 회고 로그

> 사이클: **추천(roadmap) → 수행 → 회고**.
> 아래 형식으로 한 항목씩 기록하세요.

## 템플릿
```
### YYYY-MM-DD — [훈련 주제]
- 추천: (무엇을 해보기로 했는지)
- 수행: (실제로 어떻게 했는지, 횟수/시간)
- 반응: (뽀숑이 반응, 성공/어려움)
- 회고: (잘된 점, 다음에 바꿀 점)
- 다음 단계: (이어서 할 것)
```

---

### 2026-06-08 — 이름 인지 / 이리와(Come) 시도
- 추천: (로드맵 3개월 항목) 이름 부르고 쳐다보면 즉시 보상
- 수행: 이름 부르기 + "이리와" 불러보기
- 반응: 이름 부르면 잘 쳐다봄(성공) / "이리와"는 아직 알아듣지 못함
- 회고: 이름 인지는 안정적으로 자리잡음. "이리와"는 4개월 항목이라 아직 이른 편 — 짧은 거리에서 부르며 다가올 때마다 간식 보상으로 연결하는 연습부터 시작하면 좋을 듯
- 다음 단계: 가까운 거리에서 "이리와" + 즉시 보상 짧게 반복 (4개월 본격 훈련 전 워밍업)
